# RentEase 🏠

A React-based web application for renting furniture and appliances with a modern UI, cart system, and admin dashboard.

---

## 🚀 Features

- Browse furniture & appliances  
- Add to cart with flexible rental tenure  
- Rental tracking dashboard  
- Maintenance request system  
- Admin panel with basic analytics  

---

## 🧰 Tech Stack

- React (Vite)  
- Tailwind CSS  
- Lucide React  
- React Hooks  

---

## ⚙️ Setup

```bash
git clone https://github.com/darshaningalagi/rent-easy.git
cd rentease-app
npm install
npm run dev
